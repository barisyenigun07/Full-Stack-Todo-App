# Full-Stack-Todo-App
Full Stack Todo App with Spring Boot + React

Full Stack Todo App is a basic application which fulfills CRUD operations with Spring Boot + React. 
During development, several technologies like Spring Boot, Spring Data JPA, PostgreSQL, Axios, React, React Router and Material UI are used.

When the program is run first, there comes a list of todos to screen
<img width="1792" alt="Screen Shot 2022-12-21 at 16 50 36" src="https://user-images.githubusercontent.com/61095643/208925174-c7103622-cebc-4a66-bda3-ee9de27ca0c4.png">

As it can be seen from the image, IDs, contents and status of todos are shown as table. For each row, there are also UPDATE and DELETE buttons.

To add a new todo, the user should click on the ADD TODO button. After that, there comes a page to post a todo.
<img width="1792" alt="Screen Shot 2022-12-21 at 16 51 13" src="https://user-images.githubusercontent.com/61095643/208926028-90a6d66b-740f-444d-a02f-7b964fe252ed.png">

There is a text field to write new todo and there are two buttons like POST and GO BACK below it. After the user types his/her new todo, he/she can post the todo by clicking on the POST button. To go back to the todo screen, the user should click on the GO BACK button.
<img width="1792" alt="Screen Shot 2022-12-21 at 16 51 29" src="https://user-images.githubusercontent.com/61095643/208926857-4b426afb-00f9-443e-9ce0-ac1f2423c2c5.png">

On todo table, there are two buttons like UPDATE and DELETE. When the user clicks on UPDATE button, if the status of the todo is ON PROGRESS, it becomes COMPLETED. To delete a todo, the user should click on DELETE button.
